# useDataset

A hook to access the nearest Dataset from context if no arguments are supplied, or to request a Dataset from given IRI.

Should be located within a `SessionProvider` if using to fetch.

```
const { thing, error } = useDataset(datasetIri, options)
```

## Parameters

- `datasetIri`: (optional) a dataset iri to fetch from
- `options`: (optional) additional options to pass to the dataset request, if `datasetIri` is supplied

## Return Values

- `dataset`: the Dataset retrieved from context or request using given iri (undefined if loading/missing)
- `error`: the error thrown by the request (or undefined)
