# useDataset

A hook to access the nearest Dataset from context if no arguments are supplied, or to request a Dataset from given IRI.

Should be located within a `SessionProvider` if using to fetch.

```
const { thing, error } = useDataset(datasetIri, options)
```

## Parameters

- `datasetIri`: (optional) a [Dataset](https://docs.inrupt.com/developer-tools/javascript/client-libraries/reference/glossary/#term-SolidDataset) [IRI](https://docs.inrupt.com/developer-tools/javascript/client-libraries/reference/glossary/#term-IRI) to fetch from
- `options`: (optional) additional options to pass to the dataset request, if `datasetIri` is supplied

## Return Values

- `dataset`: the Dataset retrieved from context or request using given iri (undefined if loading/missing)
- `error`: the error thrown by the request (or undefined)

## Example Usage

This example uses the datasetIri value to fetch a dataset, and pass to a child component when loaded.

<Source
  code={`
import { useDataset } from "@inrupt/solid-ui-react";
import Profile from "../components";
function ProfileLoader() {
  const datasetIri = "https://example.com/profile";
  const { dataset, error } = useDataset(datasetIri);
  if (error) return <div>failed to load</div>;
  if (!dataset) return <div>loading...</div>;
  return <Profile profileDataset={dataset} />;
}
`}
/>
