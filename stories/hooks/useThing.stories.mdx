# useThing

A hook to access the nearest Thing from context if no arguments are supplied, or to request a Thing from given iris.

Should be located within a `SessionProvider` if using to fetch.

```
const { thing, error } = useThing(datasetIri, thingIri, options)
```

## Parameters

- `datasetIri`: (optional) a dataset iri to fetch the Thing from, if both `datasetIri` and `thingIri` are supplied. Uses the Dataset from context if omitted.
- `thingIri`: (optional) a thing iri to fetch the Thing from. Uses the Thing from context if omitted.
- `options`: (optional) additional options to pass to the dataset request, if `datasetIri` is supplied

## Return Values

- `thing`: the Thing retrieved from context or requests using given iris (undefined if loading/missing)
- `error`: the error thrown by the request (or undefined)
