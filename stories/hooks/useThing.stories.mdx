# useThing

A hook to access the nearest [Thing](https://docs.inrupt.com/developer-tools/javascript/client-libraries/reference/glossary/#term-Thing) from context if no arguments are supplied, or to request a [Thing](https://docs.inrupt.com/developer-tools/javascript/client-libraries/reference/glossary/#term-Thing) from a given IRI.

Should be located within a `SessionProvider` if using to fetch.

```
const { thing, error } = useThing(datasetIri, thingIri, options)
```

## Parameters

- `datasetIri`: (optional) a [Dataset](https://docs.inrupt.com/developer-tools/javascript/client-libraries/reference/glossary/#term-SolidDataset) [IRI](https://docs.inrupt.com/developer-tools/javascript/client-libraries/reference/glossary/#term-IRI) to fetch the [Thing](https://docs.inrupt.com/developer-tools/javascript/client-libraries/reference/glossary/#term-Thing) from, if both `datasetIri` and `thingIri` are supplied. Uses the Dataset from context if omitted.
- `thingIri`: (optional) a thing iri to fetch the Thing from. Uses the Thing from context if omitted.
- `options`: (optional) additional options to pass to the dataset request, if `datasetIri` is supplied

## Return Values

- `thing`: the [Thing](https://docs.inrupt.com/developer-tools/javascript/client-libraries/reference/glossary/#term-Thing) retrieved from context or requests using given [IRI](https://docs.inrupt.com/developer-tools/javascript/client-libraries/reference/glossary/#term-IRI) (undefined if loading/missing)
- `error`: the error thrown by the request (or undefined)
